# Aditi Gupta's Portfolio Project

Welcome! This is the local workspace directory for your new portfolio website.

## How to add your resume and assets:
1. Place your resume PDF in this folder and name it `resume.pdf` (or `resume.docx`/`resume.txt`).
2. If you have any profile images, icons, or project screenshots, place them in an `assets` folder here.
3. Paste your resume details or any specific content you want on your landing page in the chat.
